# E-commerce Recommendation Analysis

## Objective
Analyze user purchase behavior and identify product associations using Market Basket Analysis.

## Tools
- Python
- Pandas
- Mlxtend
- Matplotlib

## Key Steps
- Data cleaning
- Exploratory data analysis
- Apriori algorithm
- Association rule mining

## Key Insights
- Strong product associations identified
- Opportunities for cross-selling and bundling

## Results
Association rules generated using lift and confidence metrics.
